# VideoMasker

**Secure and convenient Client-Side Video Automatic De-Identification Program**

## Requirements

Developed in `Python 3.11`
```text
pyside6
pytorch
torchvision
transformers
scikit-learn
opencv-python
huggingface-hub
imgbeddings
ultralytics
scipy
supervision
```

## Getting Started

```bash
git clone https://github.com/dh031200/VideoMasker
pip install -r requirements.txt
python main.py
```
